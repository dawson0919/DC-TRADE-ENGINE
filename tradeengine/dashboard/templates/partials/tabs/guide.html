快速上手教學

1

設定 Pionex API Key

前往「帳戶」分頁，輸入你的 Pionex API 金鑰。

  • 登入 Pionex API 管理頁面
  • 建立 API，勾選 Reading + Trading
  • 到「帳戶」分頁貼入金鑰
API Secret 只會在建立時顯示一次，請務必妥善保存。
2

回測驗證策略

先用回測確認策略表現。重點指標：夏普比率 > 1、最大回撤 < 20%、獲利因子 > 1.5

3

建立交易機器人

建議先用「模擬交易」模式測試，確認穩定後再切換「即時交易」。

即時交易使用真實資金，請務必設定停損（建議 2-5%）。
4

TradingView Webhook 整合

你可以用 TradingView 的策略信號來觸發交易，不需要使用內建策略。

  • 建立機器人時選擇「TradingView Webhook」作為信號來源
  • 建立後會取得 Webhook URL 和 Alert Message 模板
  • 在 TradingView 建立 Alert，將 Webhook URL 貼入，Message 貼入模板
  • 啟動機器人後，TradingView 觸發 Alert 時會自動執行交易

支援的 action：buy（做多）、sell（做空）、close（平倉）

5

信號跟單 (Pionex Signal Bot)

讓你的策略信號自動發送到 Pionex，跟單者可以複製你的合約交易。

Step A：在 Pionex 建立 Signal Bot

  • 登入 Pionex 交易機器人頁面
  • 點擊「建立機器人」→ 選擇「Signal Bot」（信號機器人）
  • 選擇合約交易對（例如 BTC/USDT 永續合約）
  • 設定槓桿倍率、投入金額等參數
  • 建立完成後，進入 Signal Bot 設定頁面

Step B：取得 Signal Type ID

  • 在 Signal Bot 設定頁面找到「Signal Type」或「API 信號設定」區塊
  • 複製 Signal Type ID（UUID 格式，例如：7cd6d968-15f2-4557-a499-30e30603e180）
  • 這個 ID 是你的專屬信號通道，用來接收外部策略信號

Step C：在 TradeEngine 建立信號跟單機器人

  • 前往「實盤機器人」分頁，點擊「+ 新增機器人」
  • 信號來源選擇「信號跟單 (Signal Bot)」
  • 貼入你的 Signal Type ID
  • 選擇策略、交易對、時間框架等參數
  • 啟動機器人 — 策略產生買賣信號時會自動發送到 Pionex

信號邏輯說明

  • 開多：action=buy，position_size 為正值
  • 平多：action=sell，position_size=0
  • 開空：action=sell，position_size 為負值
  • 平空：action=buy，position_size=0
  • 每次信號以 100 USDT 為基準，Pionex 會按比例分配跟單者的資金
Signal Bot 為即時交易模式，信號會直接影響跟單者的真實資金。請務必先用模擬交易驗證策略穩定性。

策略簡介

共 15 種內建策略，分為四大類：

📈 趨勢追蹤 — 適合單邊上漲/下跌行情

  • ma_crossover：均線交叉。快線穿越慢線判斷趨勢方向。
  • macd：MACD 交叉。MACD 線穿越訊號線進出場。
  • donchian：唐奇安通道突破。價格創新高/低做多/做空。
  • supertrend：SuperTrend 趨勢追蹤。基於 ATR 動態止損線翻轉。
  • vwap_crossover：VWAP 交叉。價格穿越成交量加權平均價觸發信號，可加標準差帶及 EMA 過濾。
  • triple_ema：三刀流。三條 EMA 同向排列進場。
  • turtle_breakout：海龜突破。偵測樞紐高低點突破進場。
  • dc_turtle：[DC] 海龜停利停損。樞紐突破進場 + 停利/停損/移動停利三重出場。
  • granville_pro：葛蘭碧大師。EMA 交叉 + 停損偏離。

🔄 震盪回歸 — 適合盤整行情

  • rsi：RSI 超買超賣。超賣做多、超買做空。
  • bollinger：布林通道。價格觸及通道邊界反向操作。

🔗 複合指標 — 多重確認降低假信號

  • combined：EMA 趨勢 + RSI 過濾 + MACD 確認。
  • donchian_supertrend：唐奇安通道突破 + SuperTrend 趨勢過濾，雙重確認減少假突破。
  • ichimoku_cloud：一目均衡表雲圖突破 + EMA 200 趨勢濾網，多重確認提高信號品質。

🏀 動能突破 — 連續 K 線動能判斷

  • consecutive_breakout：多空區域連續 K 線突破。

詳細參數與說明請前往「策略」分頁查看。
